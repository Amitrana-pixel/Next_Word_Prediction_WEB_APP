## 📌 Next Word Prediction using LSTM

This project implements a Next Word Prediction system using Long Short-Term Memory (LSTM) neural networks. The model is trained on textual data to learn sequential patterns and predict the most probable next word given a sequence of words.

### 🚀 Features
- Built using LSTM (Recurrent Neural Network)
- Trained on text dataset for sequence learning
- Achieves approximately **73% accuracy**
- Tokenization and padding for sequence processing
- Predicts next word based on input text

### 🧠 How It Works
The model processes input text by converting words into sequences using tokenization. These sequences are then fed into an LSTM network, which captures long-term dependencies in the text. Based on learned patterns, the model predicts the most likely next word.

### 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- NLP techniques (Tokenization, Padding)

### 📊 Model Performance
- Accuracy: ~73%
- Evaluated on validation dataset
